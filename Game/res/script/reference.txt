Variables
==========

First byte - opcode

In second byte
	First bit - value (0)/variable (1)
	
	Next two bits - data type
		00 - int
		01 - float
		10 - boolean
		11 - object
	

Third/Fourth bytes - Line number

Last four bytes - data (value/id)
